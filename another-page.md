## Python syntax

- List
```python
[2,3,5,7]
["L", "i", "s", "t"]
["This", "is", "a", "list"]
```

- Tuple
```python
(1, 2, 3, 4)
("T", "u", "p", "l", "e")
("This", "is", "a", "tuple")
```

- For
```python
for_list = ["F", "o", "r"]
for letter in for_list:
  print(letter)
```

- While
```python
counter = 0
while counter < 10:
  print(counter)
  counter += 1
```

- Function
```python
def sum(number1, number2):
  return number1 + number2
```
(Functions describe everything!)

- Dunders
```python
def main():
  pass

if __name__ == "__main__":
  main()
```
